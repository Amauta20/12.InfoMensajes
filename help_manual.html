¿Qué es InfoMensajes Power?

Un ecosistema de productividad local que reúne notas, Kanban, Gantt, Pomodoro, IA y más, todo bajo tu control y sin enviar datos a la nube.

Protocolos internos

Datos Unificados

Todo se guarda en una base SQLite local; los módulos comparten la misma fuente de datos (p. ej. Kanban ↔ Gantt).

Interacción Directa

  • Drag & Drop en Kanban.
  • Atajo global Ctrl + P para búsqueda instantánea.
  • Notificaciones de escritorio para recordatorios y eventos.

Conexión Externa

  • APIs para RSS y IA.
  • Contenedores Web seguros para Gmail, WhatsApp, etc.

Catálogo de Herramientas

Gestor de Notas

Qué es

Bloc de notas rápido y sin distracciones, pensado para capturar ideas, fragmentos de código o borradores.

Cómo se usa

  • Click en Nuevo para crear una nota.
  • Doble‑clic sobre una nota existente para editarla.
  • Usa la Búsqueda Global (Ctrl + P) para encontrar texto dentro de todas las notas.

Checklist

Qué es

Listas simples de tareas que puedes marcar como completadas.

Cómo se usa

  • Escribe una tarea y pulsa Enter para añadirla.
  • Marca la casilla para completarla.
  • Arrastra tareas entre listas independientes si lo deseas.

Tablero Kanban

Qué es

Gestión visual de proyectos mediante columnas (Pendiente, En Progreso, Hecho) y tarjetas.

Cómo se usa

  • Crea columnas y tarjetas con el botón +.
  • Arrastra tarjetas entre columnas para cambiar su estado.
  • Haz doble‑clic en una tarjeta para editar detalles (fecha, etiquetas, descripción).
[Captura de Kanban]

Diagrama de Gantt

Qué es

Vista de línea de tiempo que muestra la duración y dependencias de las tareas del Kanban.

Cómo se usa

  • Se genera automáticamente a partir del Kanban.
  • Zoom y desplazamiento para explorar periodos de tiempo.
  • Haz clic en una barra para ver o editar la tarea asociada.

Recordatorios

Cómo se usa

Temporizador Pomodoro

Qué es

Sesiones de foco de 25 min (configurable) con descansos cortos y largos.

Cómo se usa

  • Presiona Iniciar para comenzar una sesión.
  • Al terminar, se muestra una notificación y se inicia automáticamente el descanso.
  • Configura la duración en Ajustes → Pomodoro si lo deseas.

Reproduce música local para ambientar tu trabajo.

Bóveda de Seguridad

Qué es

Almacena contraseñas y secretos cifrados con una contraseña maestra.

Cómo se usa

  • Desbloquea la bóveda introduciendo la contraseña maestra.
  • Click en Añadir Secreto para guardar una nueva entrada.
  • Copiar al portapapeles o editar entradas existentes.
¡Seguridad Máxima! La contraseña maestra es la única llave; si la pierdes, no hay recuperación.

Módulos añadidos en la fase 3 (Automatización)

En la fase 3 se incorporaron tres bloques clave que amplían la capacidad de automatizar y personalizar tu flujo de trabajo.

Reglas Inteligentes

Motor de reglas "Si pasa X, haz Y". Permite crear disparadores (p. ej. fin de Pomodoro, creación de nota) y asociar acciones como show_notification, open_url o play_sound.

  1. Abre Configuración → Reglas Inteligentes.
  2. Haz clic en “Nueva Regla”.
  3. Selecciona un Disparador (ej. pomodoro_finished).
  4. Elige una Acción (ej. show_notification).
  5. Configura los parámetros (mensaje, URL, archivo de sonido) y guarda.

Ejemplo: notificar "¡Pomodoro completado!" con un sonido personalizado.

[Captura del editor de reglas]

Gestor de Plantillas (Quick Replies)

Almacena fragmentos de texto reutilizables (respuestas de correo, snippets de código, mensajes frecuentes). Cada plantilla tiene Título y Contenido.

  1. Abre Configuración → Plantillas.
  2. Presiona “Nueva” y escribe título + contenido.
  3. En cualquier parte de la aplicación usa el botón “Copiar al Portapapeles” para pegar la plantilla.

Ideal para respuestas rápidas en chats integrados o para insertar bloques de código en notas.

[Captura del gestor de plantillas]

Tablero de Métricas

Panel de productividad que muestra cuánto tiempo dedicas a cada módulo y la tendencia semanal.

  • Desglose Diario: barras horizontales por herramienta.
  • Tendencia Semanal: barra vertical con los últimos 7 días.

Acceso: botón “Métricas” en la barra superior.

Guía rápida: ¿Cómo…?

... encontrar una nota escrita la semana pasada?

Usa Búsqueda Global (Ctrl + P) y escribe una palabra clave.

... mover una tarea Kanban a "Hecho"?

Arrastra la tarjeta a la columna **Hecho**.

... guardar una contraseña de forma segura?

Abre la Bóveda, desbloquéala y pulsa **Añadir Secreto**.

... añadir mi cuenta de Outlook?

En **Ajustes → Servicios** elige **Outlook**, inicia sesión y listo.

... crear una regla para notificar al terminar el Pomodoro?

Sigue los pasos de la sección Reglas Inteligentes y elige pomodoro_finished como disparador.

... usar una plantilla de respuesta rápida en un chat?

Abre el Gestor de Plantillas, copia al portapapeles y pega en el campo del mensaje.
